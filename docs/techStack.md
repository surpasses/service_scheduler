Backend: Node.js + Express.js
Frontend: React + Vite + tailwind
Database: PostgreSQL
Testing: Vitest

<ins>Decisions and Tradeoffs:</ins>

- Decided to go with Node.js + Express.js as I am comfortable in those frameworks along with the React frontend. Decided to go with PostgreSQL due to the presence of relationships between different schemas, which is unfitting for a NoSQL database.
